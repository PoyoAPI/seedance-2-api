# Prompt Examples

These prompts are starting points. Treat them as product fixtures, then adapt them to your own UI controls, moderation rules, and customer input.

## Product Teaser

```text
A short product teaser for a matte black electric kettle. Steam rises slowly, morning light enters from the window, camera pushes in, subtle kitchen ambience
```

## Social Clip

```text
A 9:16 streetwear campaign clip. Model walks through a rainy neon crosswalk, jacket fabric moves in the wind, camera tracks from the side, energetic but realistic motion
```

Use the reference product image as the hero object. Rotate around it slowly on a minimal studio set, add soft reflections, keep brand details readable, no text overlays

## Prompt Hygiene

- Keep customer-private data out of examples, screenshots, and public issues.
- Store the final prompt version with the task ID for debugging.
- For reference media workflows, validate file type, size, and access before submit.
